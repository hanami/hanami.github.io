Announcing Lotus v0.6.0
 
We released a new version of Lotus for you. ❤
 
Today we made available a new set of features for assets.

We now have robust helpers to build complex assets markup, starting from simple Ruby code. A long list of preprocessors including Sass, Less, Babel for ES6 and JSX (React.js), CoffeScript, Opal ect..

Compressors that make possible to use powerful tools like YUI, UglifyJS2, Google Closure Compiler and others.

Fast deploys that in a few seconds precompile, compress and make assets cacheable by browsers. Deployments now support Heroku platform natively.

We can accelerate our web applications by distributing our assets via Content Delivery Networks (CDN).

And that's not all!

We shipped minor features like, custom application initializers, new Rake tasks, a "destroy" command and other small fixes.
