Hanami v0.9.0: database automapping, data integrity for entities,
experimental associations, and native PostgreSQL types.
We rewrote hanami-model from scratch with an engine based on ROM.
All the mapping boilerplate is gone, now using entities and repositories is much easier. We shipped an automatic, extra layer of protection for entities data integrity. The long time awaited associations are now available as an experimental feature.
PostgreSQL data types are now supported natively.
