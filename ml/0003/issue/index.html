Almost There
We are almost there with the release of Lotus. While waiting for the first version, I remember that you can use all the other composable frameworks in your existing applications.

This month I open sourced Lotus::Model, a persistence framework for Lotus. Apart from implementing the Data Mapper and Repository patterns for complex data manipulations, it can communicate with several data sources and it can store plain Ruby objects!

People are getting more and more excited about Lotus. Please help me to spread the word, by twetting, sharing this mailing list and hacking with the code.

Luca Guidi
